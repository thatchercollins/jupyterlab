Class ModuleLoader

A module loader using semver for dynamic resolution of requires. A namespace for ModuleLoader statics.

It is meant to be used in conjunction with the JupyterLabPlugin for WebPack from @jupyterlab/extension-builder.

Private _findMatch

  • _findMatch(path: string): string
    • Defined in application/loader.ts:169

    Find a module path matching a given module request.

    Parameters

    • path: string

      The semver-mangled fully qualified path to the module. For example, "foo@^1.1.0/lib/bar/baz.js".

    Returns string

    The matching defined module path, if registered. A match is the registered path that maximally satisfies the semver range of the request.

    Notes

    If the path has loaders, and thus multiple packages and modules delimited by '!', then the versions are matched in reverse order.
